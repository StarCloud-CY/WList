# WList

这是一个能绑定IP的白名单插件，当玩家IP改变时它将阻止玩家进入除非解除绑定
这是我的第一个插件，可能写的很差，请谅解

主指令: `/wlist`

可用子指令：

> /wlist add <玩家名> // 添加一个白名单

> /wlist del <玩家名> // 删除一个白名单

> /wlist list [页码] // 查看绑定的用户名及IP，每页10个内容

> /wlist unbind <玩家名> // 解除一个玩家对应的IP的绑定
